How do I optimize a React component that re-renders too often?
A
You can use `React.memo` to wrap the component. This prevents re-renders if the props haven't changed. Additionally, ensure that you are not passing new object references as props on every render by using `useMemo` or `useCallback`.

What is the difference between useEffect and useLayoutEffect?
A
`useEffect` runs asynchronously after the render is committed to the screen. `useLayoutEffect` runs synchronously immediately after DOM mutations but before the browser has a chance to paint. Use `useLayoutEffect` if you need to measure DOM elements or make changes that should be visible immediately to avoid flicker.

Explain the concept of "lifting state up".
A
Lifting state up involves moving the state from a child component to a common ancestor (parent) component. This allows multiple child components to share and synchronize the same state by receiving it via props.
